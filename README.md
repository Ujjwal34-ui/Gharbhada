# Ghar Bhada Management System

A responsive Apartment Rent & Utility Management System built using PHP and MySQL.

## Features
- Flat Management
- Tenant Management
- Monthly Rent Tracking
- Electricity Bill Calculation
- Water Bill Management
- Payment History
- Total Collection Tracking
- Responsive Modern UI

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
